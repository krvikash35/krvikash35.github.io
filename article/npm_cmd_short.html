NPM NOTES AND SHORTCUT

NPM Basic

Requirement Solution
Initialize empty project npm init
Install package and save to json as dependency npm install [npm_package] --save
Install package and save to json as dev-dependency npm install [npm_package] --save-dev
Install specific version of package npm install [npm_package]@[version]
Run npm in debug/verbose mode npm install [npm_package] --d

NPM Publish

Requirement Solution
set name before publishing npm set init.author.name "Your Name"
set email before publishing npm set init.author.email "me@example.com"
set url name before publishing npm set init.author.url "mywebsite.com"
login before publishing npm add user
publish the package npm publish ./
update the version before re-publishing modification to package npm version [minor/major/patch..etc]
