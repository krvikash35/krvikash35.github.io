GIT NOTE AND SHORTCUT

GIT BASIC

Requirement Solution
Configure git username git config --global user.name [name]
Configure git email git config --global user.email [email]
Initialize git local repo git init
Clone remote repo to local repo git clone --depth 1 git_repo
Add specific file or all files to staging area git add [file_name]/[.]
Remove specific file or all files to staging area git checkout [file_name]/[.]
Commit the changes git commit -m [message]
stage modified files and commit changes(ignore new files) git commit -a
push local repo to remote repo git push [remote_name] [branch]
pull remote repo to local repo with rebase git pull --rebase [remote_name] [branch]
check git status git status

GIT REMOTE

Requirement Solution
see remote repo url detail git remote -v
add remote name to give url git remote add [remote_name] [url]
remove remote repo git remote remove [remote_name]
rename remote repo git remote rename [old_remote_name] [new_remote_name]
reset remote repo url to given remote name git remote set-url [remote_name] [new_url] [old_url]
add remote repo url to given remote name git remote set-url add [remote_name] [new_url]
delete remote repo url to given remote name git remote set-url delete [remote_name] [url]
