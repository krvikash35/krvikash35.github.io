GIT NOTE AND SHORTCUT

Requirement Solution
Configure git username git config --global user.name [name]
Configure git email git config --global user.email [email]
Initialize git local repo git init
Clone remote repo to local repo git clone --depth 1 git_repo
add specific file or all files to staging area git add [file_name]/[.]
remove specific file or all files to staging area git checkout [file_name]/[.]
