Transition Transformation Animation

Isn't' it beautifull, if we can bring the motion to html element, if we can control the smoothness or speed of these motion. Such thing can create a beautifull experience for user. these are the thing that is accomplised by CSS3 feature Transition, Transformation and Animation. But how they are defined in CSS, how they work and how to use them! we will learn about these.

Transition allow us to control how fast or smooth an css property of an element will change. Transformation allow us to change the physical properties(size, shape etc) of element. Animation are like superset of transition, it provides ability to change the style of an element with multiple state, whereas transition only control how fast or smooth the sytle will change with two state(start and end).

Transition


/* Transition example: transition the change in width of div linearly in 2 sec with 1 sec start delay. */
div {
    width: 100px;
    height: 100px;
    transition-property: "width";
    transition-duration: 2s;
    transition-timing-function: "linear";//ease,linear,ease-in,ease-out,ease-in-out
    transition-delay: 1s;
}
div:hover {
    width: 300px;
}
/*   transition: width 2s linear 1s; shorthand properties        */

Transtion are triggered whenever browser detect the change in value of transition-property. transition-property for that html element allow us to choose which css property of an html element will have transition effect. ie. transition only width of div element. transition-duration tells in how much time this transition has to be completed. transition-timing-function controls how fast or slow these transition will run. there are various pre-defined function. transition-delay will delay the start of transition from actual start.

Transformation


/* Transformation example: rotate div element by 7degree */
div {
    transform: rotate(7deg);//translate(x,y) scale(x,y) skew(x-angle,y-angle)
}
/* there are other function with little change in name and parameter. i.e translate3d(x,y,z), translateX(x) */

This feature allow us to make an specific set of change in the physical size or appearence of an html element. Just attach the transform css property to an html element to which you want to transform. transform takes a specific set of built-in function. It get triggered as soon as page load.

Animation

            
/* Animation example: change the background-color of div from red to yellow in 4 second. */
@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}
div {
    width: 100px;
    height: 100px;
    background-color: blue;
    animation-name: example;
    animation-duration: 4s;
}
/* other keyframes example
@keyframes example {
    0%   {background-color: red;}
    25%  {background-color: yellow;}
    50%  {background-color: blue;}
    100% {background-color: green;}
}*/

Animation helps us to apply the different set of style at different time to an html element. Animation start immediatly as soon as page load. first we decide, which element we want to animate by attaching css propery animation-name. Then we define the animation with keyword @keyframes
