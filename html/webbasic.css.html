Transition Transformation Animation
These three features are part of CSS3. All three are different to each other but having one common thing 'motion'. so due to this common thing, sometimes they are used together or interchangeably.
Transition allow us to control how fast or smooth an css property of an element will change. Transformation allow us to change the physical properties(size, shape etc) of element. Animation are like mix of transition and Transformation.
            
div {
    width: 100px;
    height: 100px;
    transition-property: "width";
    transition-duration: 2s;
    transition-timing-function: "linear";//ease,linear,ease-in,ease-out,ease-in-out
    transition-delay: 1s;
}
div{
    transition: width 2s linear 1s;//shorthand properties
}            
        
            
div {
    transform: rotate(7deg);
}
            
        
            
@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}
div {
    width: 100px;
    height: 100px;
    background-color: red;
    animation-name: example;
    animation-duration: 4s;
}
